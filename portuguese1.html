Objective

Develop a responsive, interactive web application that serves as an AI-powered Portuguese language tutor tailored for English-speaking users. The application features a Language Model interface and console optimized for audio input and output, alongside text-to-text interactions. Users can upload attachments (e.g., images of text, conversational media) to enhance the learning experience. Outputs are delivered in text with an optional toggle for audio output using high-quality voice synthesis.

Core Technologies

  • Language Models: Utilize GPT-4-turbo for interactive tutoring and Whisper for accurate audio input processing.
  • Voice Synthesis: Implement Eleven Labs or a similar high-fidelity voice synthesis API for audio outputs.
  • Translation Services: Integrate Google Translate for contextual accuracy and DeepL for nuanced translations.
  • Additional APIs: Incorporate language learning APIs that offer grammar and pronunciation scoring to enhance feedback mechanisms.

Key Features

1. LLM Interface and Console

  • Optimized Audio Input/Output: Users interact via voice commands using a microphone. Real-time transcription of audio inputs using Whisper. Audio outputs with high-quality voice synthesis.
  • Text-to-Text Interaction: Maintain traditional text input/output for users who prefer typing.

2. Vocabulary Building Modules

  • Structured Progression: Begin with the most essential 100 Portuguese words, advancing to 250 and then 500.
  • Interactive Learning: Each vocabulary set includes contextual examples, sentences, phrases, and interactive exercises.
  • Customizable Lists: Users customize vocabulary lists based on specific interests or professional needs.
